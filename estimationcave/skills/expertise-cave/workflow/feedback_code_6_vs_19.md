description: Pour les vins primeurs Bordeaux à prix accessibles (Cru Bourgeois, Bordeaux régional, satellites) sans dimension spéculative, utiliser code 6 (à conserver, valeur dégustation > revente) plutôt que code 19 (à surveiller, primeur).

Pour les primeurs Bordeaux à prix accessibles (Cru Bourgeois, Bordeaux régional, Côtes de Bordeaux, satellites de Saint-Émilion) sans dimension de marché secondaire, recommander **À conserver code 6** et non À surveiller code 19.

**Why:** Le code 19 (primeur, marché secondaire en formation) est réservé aux grands crus avec dynamique spéculative (Léoville-Poyferré 2023, etc.) où on attend la formation du marché secondaire. Pour les primeurs vocation-table (Marjosse 9€, Lilian Ladouys 16€, Montlandrie 25€, Potensac 25€, Larrivaux 17€), il n'y aura jamais de marché secondaire — la "valeur" est dans la dégustation. Donc code 6 est le bon code, et la reco est À conserver pour consommation.

**How to apply:**
- Primeur grand cru classé / signature spéculative → code 19, À surveiller, date de réexamen 2027-2028
- Primeur Cru Bourgeois / Bordeaux régional / satellite → code 6, À conserver, durée de garde sur logique table (8-15 ans), pas de date de réexamen
- Le pivot se fait autour de 30-40€/btl primeur : au-dessus, dimension spéculative possible ; en dessous, vocation plaisir.

Exception déjà validée : Larrivaux 2023 (17€) reste en code 19 dans le batch 252-260 — Fanny a validé sans correction. Mais à partir du batch 261-270, elle a explicitement réorienté Lilian Ladouys, Marjosse, Montlandrie et Potensac vers code 6. Pattern stable : par défaut, primeur accessible = code 6.
