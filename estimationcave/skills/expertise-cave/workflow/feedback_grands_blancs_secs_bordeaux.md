Les grands Bordeaux blancs secs issus de domaines de Sauternes ou de Premiers Crus Classés (Pavillon Blanc de Château Margaux, Aile d'Argent de Mouton-Rothschild, Y d'Yquem, Suduiraut Vieilles Vignes Blanc Sec, etc.) constituent une **catégorie de prestige avec marché secondaire actif**. Recommander **code 1 ou 4** (à conserver) selon la dynamique du millésime, **pas code 6**.

**Why:** Ces blancs sont rares (production confidentielle), ont une garde longue (10-15+ ans), et bénéficient d'une critique presse haut de gamme régulière (Galloni, Dunnuck, Suckling, Martin, Ma, Beck, Wine Spectator). Les amateurs avertis les recherchent activement aux enchères. Le code 6 (valeur dégustation > revente) sous-estime leur dimension marché. Pattern précisé après correction Fanny sur Suduiraut Blanc Sec VV 2023 (initialement code 6, corrigé en code 1).

**How to apply:**
- Pour ces signatures (Suduiraut VV, Pavillon Blanc Margaux, Aile d'Argent Mouton, Y d'Yquem, Couhins-Lurton, Smith Haut Lafitte Blanc, Domaine de Chevalier Blanc, Haut-Brion Blanc, Laville-Haut-Brion) : code 1 si avant apogée, code 4 si grand millésime confirmé, code 7 si apogée dépassée.
- Mentionner explicitement les notes critiques détaillées dans la note marché — c'est un argument de valorisation pour cette catégorie où le score presse pèse lourd.
- Liquidité Moyenne (et non Faible) — les amateurs avertis structurent un marché stable.
- Pour les blancs secs de Bordeaux sans cette dimension prestige (Bordeaux blancs régionaux, Entre-deux-Mers, etc.), code 6 reste valable.
