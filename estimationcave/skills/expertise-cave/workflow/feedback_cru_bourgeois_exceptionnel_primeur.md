Un Cru Bourgeois **Exceptionnel** (la classe la plus haute du classement Cru Bourgeois) en primeur, dès que la Val_unit atteint 25€+, doit être recommandé en **code 19 (primeur, à surveiller)** plutôt que code 6.

**Why:** Cru Bourgeois Exceptionnel = signature qualifiée avec public d'amateurs structuré. Le marché secondaire existe (même de niche) et la dimension primeur justifie l'attente de stabilisation de cote. Code 6 reste pertinent pour Cru Bourgeois génériques ou primeurs <25€. Pattern précisé après correction Fanny sur Haut-Marbuzet 2023 (Val 25€) — initialement code 6, corrigé en code 19.

**How to apply:**
- Cru Bourgeois Exceptionnel primeur 25€+ → code 19, À surveiller, date de réexamen 2027-2028
- Cru Bourgeois Exceptionnel non primeur (millésime livré) → code 1 si avant apogée, 4 si grand millésime, 7 si apogée dépassée
- Cru Bourgeois Supérieur ou générique en primeur 15-20€ → code 6 reste valable

Cas de référence : Haut-Marbuzet (Saint-Estèphe), Lilian Ladouys (Saint-Estèphe), Phélan-Ségur, Le Boscq, Sociando-Mallet (non classé mais profil équivalent), Potensac (Médoc).
